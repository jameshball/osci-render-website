an audio plugin for audio-visual music-making


osci-render is a synthesizer for making music by drawing video, objects, text, blender animations, and images on an oscilloscope using audio output.


learn more about osci-render on my youtube channel.


join the discord server with 1000+ members for support, updates, and to share your art, or email me at james@ball.sh.

Osci-render premium gives you the following extra features, and is required for commercial use:

Video Support

Convert MP4 and MOV files into oscilloscope audio

Live Video Input

Real-time video input with Syphon / Spout support

Enhanced Simulation

More realistic analogue oscilloscope simulation

Advanced Controls

Customize framerate and resolution settings

Recording

Export your visuals as MP4 files

Enhanced Text

Improved text rendering quality

It can convert just about every multimedia file type you could imagine into audio:

  • MP4 / MOV (premium only!)
  • Live video input with Syphon / Spout (premium only!)
  • JPG / PNG
  • GIF
  • SVG
  • Text
  • 3D objects
  • Blender animations

On top of this, you can program your own synthesiser using the built-in Lua support.
