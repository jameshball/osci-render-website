osci-render

A synthesizer for making music by drawing objects, text, and images on an oscilloscope using audio output.

Overview

Synthesizer for making music by drawing objects, text, and images on an oscilloscope using audio output. This allows for 3D rendering of .obj files, .svg images, .txt files, and Blender scenes on Windows, macOS, and Linux.

For any issues, feedback, feature requests, or inquiries, contact james@ball.sh.
